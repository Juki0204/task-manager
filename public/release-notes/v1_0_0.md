数か月運用してきて、不具合などもひとしきり修正が完了し、目立った大きな不具合も今のところないので、一旦一区切りとして、これを正式版とします。<br>

## データベースのバックアップを定期的に取るようにしました。
github actionにて定時(毎日朝3時頃)に全テーブルのバックアップを取るようにしました。<br>
直近1週間分が常にストックされますので、もし取り返しのつかない破壊的データバグなどが発生しても復元可能です。<br>

## 作業ルール・情報共有掲示板を実装しました。
以前のスプシでいうところのフリースペースの代替機能となります。<br>
UI操作などは概ねタスクの方の操作感を踏襲しているので、そこまで迷うことはないかと思います。<br>
ある程度の仕様や操作方法をページ内左部に記載しているので、利用する際は一読ください。<br>
